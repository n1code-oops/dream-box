# Dream Box

开发一款 `Dream Box`，逐渐拓展完善...

## 开发计划

- 资产计算器
  - 每日产出统计
  - 资产总计
- 提醒事项
  - 允许设置提醒时间，和循环周期

  ......

## 本地开发

```bash
pnpm dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000) 查看页面。

编辑 `src/app/page.tsx` 即可实时预览改动。

## 其他命令

```bash
pnpm build   # 生产构建
pnpm start   # 启动生产服务
pnpm lint    # 代码检查
```
